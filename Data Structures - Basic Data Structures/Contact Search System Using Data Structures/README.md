# Contact Search System Using Data Structures

This is a module/checkpoint folder.
It stores files used for this part of the repository.

## Overview
A simple contact management system written in JavaScript that combines a
doubly linked list, a hash table, and a naive substring search to add,
look up, and display contacts from a text menu.

## Contents
- contactSystem.js — runnable JavaScript implementation.
- projet.md — full description, code walkthrough, and example run.

## How to Run
```bash
node contactSystem.js
```
