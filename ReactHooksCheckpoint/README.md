# React Hooks Checkpoint — Movie App

A small movie catalog built with functional components and React hooks
(`useState`). You can add new movies and filter the collection by title or
by minimum rating.

## Components

- `MovieCard` — renders a single movie (title, description, poster, rating)
- `MovieList` — renders the filtered grid of `MovieCard`s
- `Filter` — controlled inputs for title search and minimum rating
- `AddMovie` — form to append a new movie to the list

## Getting started

```bash
npm install
npm start
```

The app runs at [http://localhost:3000](http://localhost:3000).
