# Checkpoint Decision Making and Recursive Algorithms

This is a module/checkpoint folder.
It stores files used for this part of the repository.

## Overview
A single JavaScript file that solves **all six** tasks from the brief —
three on decision making (`if`/`else if`/`switch`) and three on recursion.
Decision-making tasks: **Leap Year Checker**, **Ticket Pricing**,
**Weather Clothing Adviser**. Recursion tasks: **Fibonacci** (plain +
memoised), **Palindrome Checker** (ignores spaces, punctuation, case),
**Power Function** (plain + fast-exponentiation `O(log n)` variant).
Each function ships with a small demo block.

## Contents
- decisionsAndRecursion.js — runnable JavaScript implementations and demo.
- projet.md — task-by-task walkthrough, code, sample outputs.

## How to Run
```bash
node decisionsAndRecursion.js
```
