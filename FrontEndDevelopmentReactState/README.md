# React State Checkpoint — Class Components & Lifecycle

Class-based `App` component with a `Person` state, a toggle button that shows/hides
the profile, and a live counter that displays the time elapsed since the component
was mounted (via `setInterval` in `componentDidMount`).

## Getting started

```bash
npm install
npm start
```

The app runs at [http://localhost:3000](http://localhost:3000).
