Notre Histoire

Teranga — mot wolof pour hospitalité — est né en 2008 au cœur de Dakar, dans le quartier animé du Plateau. Notre restaurant incarne l'âme du Sénégal : chaleur humaine, générosité et saveurs incomparables.

Notre cheffe, Aminata Diallo, perpétue les recettes transmises de génération en génération. Elle sélectionne chaque matin les ingrédients frais au marché Sandaga pour garantir l'authenticité de chaque plat.

Que vous soyez dakarois de cœur ou visiteur de passage, chez Teranga vous êtes chez vous. Venez découvrir la richesse culinaire d'un pays où manger ensemble est un art de vivre.
