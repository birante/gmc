# Graph Basics and Traversal

This is a module/checkpoint folder.
It stores files used for this part of the repository.

## Overview
A simple graph implementation in JavaScript that represents a graph as an
adjacency list (Map of Sets) and exposes the core operations: add edge,
remove edge, check edge, print, plus BFS and DFS traversals. Supports both
directed and undirected graphs.

## Contents
- graphTraversal.js — runnable JavaScript implementation.
- projet.md — full description, code walkthrough, and example run.

## How to Run
```bash
node graphTraversal.js
```
