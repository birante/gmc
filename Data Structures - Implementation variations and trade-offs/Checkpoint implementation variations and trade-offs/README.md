# Checkpoint: Implementation Variations and Trade-offs

This is a module/checkpoint folder.
It stores files used for this part of the repository.

## Overview
Two Queue variants (fixed-size circular array and singly-linked list) and two
Priority Queue variants (min-heap and ordered array) implemented in
JavaScript. Each structure exposes the standard operations, handles edge
cases (empty / full), and is annotated with its time and space complexity so
the trade-offs are explicit.

## Contents
- queueAndPriorityQueue.js — runnable JavaScript implementation and demo.
- projet.md — full description, code walkthrough, complexity tables, and trade-off discussion.

## How to Run
```bash
node queueAndPriorityQueue.js
```
