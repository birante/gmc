# Optimizing Task Scheduling System

This is a module/checkpoint folder.
It stores files used for this part of the repository.

## Overview
A lightweight task scheduler in JavaScript that stores tasks with start/end
times and priority, sorts them by start time, groups them by priority with a
hash map, detects overlapping intervals, and offers an optional memory
estimate. Each operation is annotated with its time and space complexity.

## Contents
- taskScheduler.js — runnable JavaScript implementation.
- projet.md — full description, code walkthrough, and example run.

## How to Run
```bash
node taskScheduler.js
```
